# screen_capture.py 使用マニュアル

**対象ファイル**: `src/capture/screen_capture.py`
**Sprint**: 1

---

## 概要

ポケモン対戦画面からテキスト（HP・ポケモン名・技名）を取得するスクリプト。
2つのモードがある。

| モード | コマンド | 実行環境 | 用途 |
|--------|---------|---------|------|
| ファイルモード | `--image` | WSL2・Windows どちらでも可 | OCRの動作確認・精度テスト |
| ライブモード | `--live` | **Windows Python のみ** | 実際のゲーム画面をリアルタイム認識 |

> **なぜライブモードはWindowsのみ？**
> Claude Code はWSL2（Linux）で動作しており、WSL2の `mss` はWindowsのゲーム画面を直接キャプチャできない。ライブモードはWindowsネイティブのPythonで実行する必要がある。

## モード2: ライブモード（リアルタイムキャプチャ）

1秒ごとに画面をキャプチャし、画面変化（ターン切替・技使用など）を検知したときだけOCRを実行する。

### コマンド

**Windowsのコマンドプロンプト / PowerShell で実行すること。**

```powershell
# 基本（1秒間隔）
python src\capture\screen_capture.py --live

# キャプチャ間隔を変更（0.5秒ごと）
python src\capture\screen_capture.py --live --interval 0.5

# CPUモードで実行
python src\capture\screen_capture.py --live --cpu
```

### 動作の流れ

```
起動
　↓
1秒ごとに画面キャプチャ
　↓
前フレームとの差分を計算（OpenCV）
　↓
差分が閾値を超えた？
　├── NO  → 次のキャプチャへ（OCRしない）
　└── YES → イベント検知！OCR実行 + スクショ保存
```

### 出力例

```
12:34:56 [INFO] ライブキャプチャ開始（間隔: 1.0秒、Ctrl+Cで終了）
12:34:56 [INFO] キャプチャ対象モニター: 1920x1080
                 ← 変化がない間はログ出力なし
12:35:12 [INFO] [ターン 1] イベント検知！ 差分スコア: 45.23 → OCR実行

==================================================
テキスト                              信頼度
--------------------------------------------------
✓ ガブリアス                          94.2%
✓ HP 92/185                           88.1%
...
==================================================

12:35:12 [INFO] スクリーンショット保存: debug_turn_001.png
```

### 停止方法

```
Ctrl + C
```

### 保存されるファイル

イベント検知ごとにスクリーンショットが自動保存される（デバッグ用）。

```
debug_turn_001.png   ← ターン1のスクショ
debug_turn_002.png   ← ターン2のスクショ
...
```

保存先はスクリプトを実行したカレントディレクトリ。

## オプション一覧

| オプション | 型 | デフォルト | 説明 |
|-----------|---|---------|------|
| `--image PATH` | 文字列 | — | OCRするファイルのパス（ファイルモード） |
| `--live` | フラグ | — | ライブキャプチャモード |
| `--interval 秒数` | 小数 | `1.0` | キャプチャ間隔（ライブモードのみ有効） |
| `--cpu` | フラグ | 無効 | GPUを使わずCPUモードで実行 |

`--image` と `--live` はどちらか一方が必須（同時指定不可）。

## よくあるトラブル

### 初回起動が遅い
EasyOCRのモデル（約500MB）を自動ダウンロードするため、初回は1〜2分かかる。2回目以降はキャッシュされて数秒で起動する。

### `No module named 'mss'` エラー
仮想環境が有効化されていない。

```bash
source venv/bin/activate   # WSL2
venv\Scripts\activate      # Windows
```

### ライブモードでキャプチャできない / 真っ黒な画像になる
WSL2で `--live` を実行している可能性がある。**Windowsのコマンドプロンプト/PowerShell で実行すること。**

### OCRが日本語を誤認識する
- 信頼度が低い（`?` マークのもの）は精度が不十分
- Sprint 5でポケモン名辞書による後補正を実装予定
- とりあえず信頼度0.5以上のみを使う実装で進める

### 差分検出でイベントが検知されない / されすぎる
`DiffDetector` クラスの定数を調整する（`screen_capture.py` 78〜79行目）。

```python
DIFF_THRESHOLD = 30    # 大きくすると検知しにくくなる（誤検知が減る）
MIN_CHANGE_AREA = 1000  # 大きくすると小さな変化を無視する
```
