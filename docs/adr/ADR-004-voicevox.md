# ADR-004: 音声合成にVOICEVOXを使用する

## ステータス
承認済み

## 日付
2026-02-24

## 文脈

実況には自然な日本語音声が必要。GPU・コスト・品質・ローカル動作の可否を考慮して音声合成ツールを選定する。3DモデルとのリップシンクにはVMC Protocolなどの連携が必要になる。

## 決定

VOICEVOXをローカルで使用する。

## 理由

- 無料で利用できる
- CPU動作のためGPUリソースを消費しない（VRAM節約）
- 日本語の音声品質が高く実況として自然に聞こえる
- ローカルAPIとして動作するためPythonから簡単に呼び出せる
- 複数のキャラクターボイスが用意されており実況キャラに合わせて選択できる

## 却下した選択肢

| 選択肢 | 却下理由 |
|--------|---------|
| Amazon Polly | 費用が発生する・感情表現が弱い・VOICEVOXより不自然 |
| ElevenLabs | 高品質だがコストが高い・外部API依存 |
| Style-BERT-VITS2 | 高品質だがセットアップが複雑 |

## 連携方法

```
実況テキスト生成
　↓
VOICEVOX API（localhost:50021）
　↓ WAVファイル生成
音声再生 ＋ 3Dモデル口パク同期（VMC Protocol等）
```

## 結果

GPU・コストを消費せず高品質な日本語音声を実現できる。Sprint 3でセットアップし3Dモデルとの口パク同期まで実装する。
